자주 묻는 질문

기존 CMS를 교체해야 사용할 수 있나요?

아니요. 기존 운영 화면은 유지하면서 API Gateway, 모니터링, 승인 흐름을 단계적으로 붙일 수 있도록 구성했습니다.

운영계 배포 전 검수 단계를 둘 수 있나요?

가능합니다. 사이트별 승인자, 예약 배포 시간, 반려 사유, 변경 비교 화면을 정책에 맞게 분리할 수 있어요.

API 사용량과 장애 알림은 어디서 보나요?

대시보드에서 요청량, 실패율, 응답 시간, 최근 장애 이력을 확인하고 임계치 기반 알림을 설정할 수 있습니다.

권한 관리는 어떻게 하나요?

서비스 계정별 토큰, 역할 기반 권한, 접근 허용 IP, 만료 정책을 나눠 관리합니다. 모든 변경은 감사 로그로 남습니다.
